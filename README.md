Student Social Media Insights Dashboard | Power BI

Project Overview
This project presents an interactive Power BI dashboard** that analyzes how students’ social media usage influences their sleep patterns, academic performance, mental
health, and relationship dynamics.The dashboard is designed to provide clear, data-driven insights for academic advisors, counselors, and researchers.

This project demonstrates practical skills required for **Data Analyst and MIS Executive roles**.

Dataset Overview
- **Source:** Excel Workbook  
- **Total Sheets:** 2  
- **Student Details Sheet:** 705 rows  
- **Platform Details Sheet:** Social media usage data  
- **Primary Key:** Student_ID  
- **Data Type:** Structured (numerical and categorical)

✔ Data cleaned and validated  
✔ Suitable for dashboarding and reporting  

 Stakeholders
- Academic Advisors – Academic performance analysis  
- Mental Health Counselors – Sleep, addiction, and mental health insights  
- Institutional Researchers – Country and gender-level trends  
- Students & Counselors – Individual student profile analysis  

 Data Modeling
- Created a **1:1 relationship** between both sheets using Student_ID  
- Implemented **Star Schema** for efficient analysis  
- Fact Table: Platform Details  
- Dimension Table: Student Details  
- Added a custom Date Table for future time-based analysis  

 Data Preparation
- Created calculated columns for:
  - Mental Health Band classification
  - Conflict Level categorization
- Built key performance measures for:
